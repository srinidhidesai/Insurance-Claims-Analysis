## Insurance Claims Analysis Dashboard

### Objective
Analyze insurance claims, coverage, and premium data to identify key risk patterns and business insights.

### About the Company
SD Insurance Pvt. Ltd. is a **fictional insurance company** created for educational and portfolio purposes.  
All data used in this project is **synthetic**.

### Key Insights
- Rejected claims exceed settled claims across most policy types.
- Travel and Health policies show the highest coverage exposure and rejections.
- Adult customers contribute the highest claim amount.
- Premium revenue is primarily driven by Travel and Health policies.

### Tools Used
- Power BI
- DAX (basic measures)
- Data Modeling
